#### Database
So in order to make database:
```html
phpMyAdmin > New Database: laradb
```

Put your database username and password in ```.env``` file. (root directory)

```html
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laradb
DB_USERNAME=root
DB_PASSWORD=

```
